# Introduction

Rangoli is a shadcn/ui-based component library designed to provide developers with ready-to-use, beautifully styled UI elements. It includes essential components like pricing tables, testimonials, and more, making it easy to build modern, elegant interfaces.

![My Image](/img/ui.avif)

## Why Choose Rangoli Over Other Libraries?
1. Prebuilt & Customizable – Get high-quality, production-ready components while keeping full control over customization.
2. Optimized for Modern UIs – Includes essential business and marketing components like pricing sections and testimonials.
3. All Components supports auto translation to multi-language, and dark/light mode.
4. Consistent & Accessible – Designed with accessibility in mind to create inclusive user experiences.
5. ShadCN-Based & Tailwind Compatible – Leverages shadcn/ui with Tailwind support for seamless styling and flexibility.
